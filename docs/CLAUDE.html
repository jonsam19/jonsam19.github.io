This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

Project Overview

This is a personal portfolio website built with Quarto, published as a GitHub Pages site. The site showcases Jonas Samuelsson’s professional work, projects, and travels.

Architecture

  • Static Site Generator: Quarto (R-based literate programming and publication system)
  • Output Directory: docs/ (configured for GitHub Pages)
  • Content Files: .qmd (Quarto markdown) files in root directory
  • Styling: Custom SCSS in styles.scss with color theme variables
  • Extensions: FontAwesome extension in _extensions/quarto-ext/fontawesome/

Key Files Structure

  • _quarto.yml - Main configuration file defining site structure, theme, and navbar
  • index.qmd - Home page with personal introduction and social links
  • projects.qmd - Portfolio of Shiny apps and data science projects
  • travel.qmd - Interactive world map showing countries visited (R code with leaflet)
  • resources.qmd - Additional resources page
  • styles.scss - Custom styling with defined color palette
  • media/ - Static assets (images, CV PDF)

Development Commands

Building the Site

quarto render

Preview During Development

quarto preview

Publishing to GitHub Pages

The site is automatically built to the docs/ directory which is served by GitHub Pages.

Content Guidelines

  • All content pages are .qmd files in the root directory
  • Images and static files go in media/ directory
  • The site uses a custom color theme defined in styles.scss:
    • Primary colors: dark green (#858E79), light green (#aac4af), cream (#FDFBF7)
    • Accent colors: purple (#9158A2), orange (#ad7237), gray (#64605f)

R Integration

The travel.qmd file contains R code that generates an interactive map using: - leaflet for interactive mapping - rnaturalearth for world country data - sf for spatial data handling - Countries visited are hardcoded in the R chunk

RStudio Project

This is configured as an RStudio project (.Rproj file) with: - UTF-8 encoding - 2 spaces for tabs - Code indexing enabled
